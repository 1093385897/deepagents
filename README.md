"# DeepAgents - 对话式多智能体研究系统

## 项目描述

DeepAgents 是一个基于大型语言模型（LLM）的对话式多智能体研究系统，旨在探索和实现复杂任务分解、多智能体协作和人机交互的先进方法。

## 核心特性

- **多智能体架构**：支持主智能体与多个专业子智能体协作，包括：
  - 数据库查询智能体
  - 知识库检索智能体
  - 网络搜索智能体
- **异步流式响应**：实时流式输出，提升用户体验
- **RAG 支持**：集成知识库检索增强生成能力
- **灵活的工具系统**：支持自定义工具调用和中间件
- **会话管理**：支持跨会话记忆和上下文管理
- **人机交互**：支持人工审批和干预机制

## 技术栈

- **后端**：Python + FastAPI
- **前端**：React + TypeScript + Vite
- **智能体框架**：支持 DeepAgent、LangGraph、LangChain
- **数据库**：MySQL + ChromaDB
- **部署**：Docker Compose"
